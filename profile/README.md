**Sobre**
- Hospedagem e entrega de streaming com experiências de alto impacto visual.
- Projetos em WordPress/PHP e front‑ends modernos com React, Vue e Next.js.
- Mobile First, acessibilidade e micro‑interações performáticas com Motion.dev.
- Atuação em múltiplas plataformas: web, iOS, Android, TVs, Roku e WebView.

**Tecnologias**
- Frameworks: React, Vue, Next.js, Vite, Tailwind CSS, Motion.dev, Radix UI, A11y, Prettier.
- Linguagens: TypeScript, JavaScript, Python, C#, C++ e padrões web.
- WordPress: temas com build Vite, assets versionados por `manifest.json` e CSR.

**Destaques do Repositório**
- Tema WordPress com front‑end em React (CSR), Vite e Tailwind 4.1.
- Integração com Motion.dev para micro‑interações leves e de alto impacto.
- Build de produção e registro de assets via `wp-theme-ecast/dist/.vite/manifest.json`.
- Montagem do app em `wp-theme-ecast/index.php` com `<div id="root"></div>`.
- Scripts úteis: `dev`, `build`, `preview` na pasta `wp-theme-ecast/react`.

**Como começar (tema WordPress + React/Vite)**
- `cd wp-theme-ecast/react` e `npm install`.
- Desenvolvimento: `npm run dev`.
- Preview de build: `npm run preview` na porta `3000`.
- Produção: `npm run build` (artefatos em `../dist/`).
